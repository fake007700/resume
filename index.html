  • Peer to many peer file copy program using IPC [Repository]

    Guide: Dr. Sambuddho Chakravarty
    Designed a file copying system using Inter-Process Communication (IPC) with two mecha- nisms using C language. The first uses domain sockets (AF UNIX) where the sender estab- lishes a listening socket, accepts client connections, and forks child processes to serve each client, transmitting the file byte by byte using write().
